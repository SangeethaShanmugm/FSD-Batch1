CSS Selectors
-----------------

----> id  - # (hash) - unique elements
 syntax
 #h1
 {
   text-align: center;
 }


-----> class - . (dot) -  group of elements
syntax
.h1{
     text-align: center;
}


CSS Box model
---------------

all HTML elements are considered as Boxes

p {
            width: 100px;
            padding: 20px;
            border: 10px solid black;
            margin: 0;
        }

100px width + 20px (left + right padding) +  10px (left + right border ) + 0 px(left +right margin)

Margin -property
